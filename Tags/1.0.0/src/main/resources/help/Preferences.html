Preferences


DeviantArt Favorites Downloader can be customized in order to better fit your sorting habits.

Username

This control which user's favorite you're downloading. For most usage, it will be your own deviantART username, but if you want to download someone's else favorites or a club favorites, you can input their username there.

Download Location

This control where the 'clean' deviations will be saved to. You can build any path as long as the drive exists and you don't use forbidden characters. You can customize path using informations from the downloaded deviation by substituting their field inside the path. The fields are listed bellow. A sample path is showed in the example field at the bottom.

Mature Download Location

This control where the 'mature' deviations will be saved to. Should you want to save those deviations elsewhere, safety, privacy, usage or any reasons, you can specify it there. If you don't mind having both clean and matures deviations saved in the same destination, input the same path or check the 'Use same location for mature deviation' box. A sample path is showed in the example field at the bottom.

Fields

  • %user% The user's username whose favorites are being downloaded. Same as inputed in the username box above,
  • %artist% The username of person who has submitted the deviation being saved,
  • %title% The deviation's title, aka the 'Deviation Title'. The title is normalized to remove any forbidden characters,
  • %id% The deviation's id, example 123456789,
  • %filename% The deviation's filename,
  • %ext% The deviation's filename extension

At least one of the supplied field must be supplied: %id%, %filename%.

Throttling

The application doesn't download files that have already been downloaded. When performing a rerun, this mean files will be skipped and deviantART will be contacted over and over for favorites updates. To lessen the load, a delay between search can be configured. If the delay between searches are less than 10 seconds, a pause is performed for as many seconds specified here.
